# Data Mining Project

## Course Overview
This folder contains projects and assignments completed as part of the
**Data Mining** course during my Master's in Data Science.

The course focused on discovering patterns, extracting knowledge from data,
and applying analytical techniques to real-world datasets.

## Topics Covered
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Pattern discovery
- Feature selection
- Model interpretation

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Folder Structure
- `docs/` → Project reports and documentation
- `notebooks/` → Jupyter notebooks
- `src/` → Python scripts
- `data_sample/` → Sample datasets (if any)
- `images/` → Visualizations and screenshots

## Outcome
This subject strengthened my understanding of how to analyze data,
extract meaningful patterns, and support data-driven decision making.
## Deliverables
- Report: `docs/`
- Presentation: `docs/`
- Code: `notebooks/` and/or `src/`
